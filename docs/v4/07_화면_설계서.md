# TicketFlow — 사용자 시나리오별 와이어프레임

> 페르소나 3인의 시나리오 흐름 순서대로 관련 화면 와이어프레임을 정리한 문서입니다.
> API 설계 검증용 — ASCII 아트 기반

---

## 페르소나 1 — 박서준 (티켓팅 전사)

> **Goal:** 원하는 좌석을 공정하게, 확실하게 선착순으로 확보하고 싶다.

---

### 시나리오 1-1 · 1-2: 콘서트 티켓 오픈 직후 좌석 선점 + 동시 경쟁

**화면 흐름:** SCR-003 홈 → SCR-005 이벤트 상세 → SCR-006 좌석 선택 → SCR-007 주문·결제 → SCR-008 결제 완료
